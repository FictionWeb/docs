---
title: FictionServePath
---
Creates a new route

```sh
FictionServeType "<route>" "<function>" "<content type>" "<route type?>"
```

| Argument | Description | Required |
| --- | --- | --- |
| $1 | Route name | true |
| $2 | Function or command to execute on route | false |
| $3 | Use custom content type instead of automatically defined one | false |
| $4 | Specify route type (static/dynamic/api/cgi/dynamic-api) | false |
